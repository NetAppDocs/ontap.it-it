= Scopri le LIF di ONTAP per gli ambienti SAN
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Le LUN e i namespace nel tuo sistema ONTAP sono contenuti all'interno delle storage virtual machine (SVM). Le interfacce di rete (LIF) consentono ai client SAN di connettersi a una SVM e accedere ai dati nelle LUN e nei namespace che essa contiene.

* È necessario configurare un LIF di gestione per ogni macchina virtuale di storage (SVM) che supporta SAN.
* È possibile configurare due LIF per nodo, una per ogni fabric utilizzato con FC e per separare le reti Ethernet per iSCSI.
* Per la connettività è supportata la connessione diretta o l'utilizzo di switch Ethernet. È necessario configurare i LIF per entrambi i tipi di connettività.
