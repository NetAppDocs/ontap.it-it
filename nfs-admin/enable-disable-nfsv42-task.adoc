= Abilitare o disabilitare NFS per SVM ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
È possibile abilitare o disabilitare le versioni di NFS tramite System Manager o la CLI. Ciò consente l'accesso ai file per i client che utilizzano il protocollo NFS.

Per impostazione predefinita, NFSv3 è abilitato in tutte le versioni supportate di ONTAP 9. A partire da ONTAP 9.9.1, NFSv4.1 e NFSv4.2 sono abilitati per impostazione predefinita; nelle versioni precedenti, sono disabilitati per impostazione predefinita.

A partire da ONTAP 9.19.1, è possibile abilitare o disabilitare NFSv4.1 e NFSv4.2 in modo indipendente. Prima di ONTAP 9.19.1, era possibile abilitare o disabilitare NFSv4.1 e NFSv4.2 solo contemporaneamente. Per impostazione predefinita, sia NFSv4.1 che NFSv4.2 sono abilitati quando viene creato un nuovo servizio NFS, a meno che non vengano esplicitamente disabilitati dall'amministratore.


NOTE: Il monitoraggio FPolicy degli eventi di accesso ai file NFSv4.2 non è supportato. Se si configura il monitoraggio FPolicy, è necessario disabilitare NFSv4.2 sulla SVM. Per riferimento, vedere link:nas-audit/supported-file-operation-filter-fpolicy-nfsv4-concept.html["Combinazioni di operazioni di file e filtri supportate che ONTAP FPolicy monitora per NFSv4"].

[role="tabbed-block"]
====
.System Manager
--
.Fasi
. In System Manager, selezionare *Cluster > Storage VM*, selezionare la Storage VM, selezionare *Impostazioni* e quindi selezionare image:icon_gear.gif["Icona delle azioni"] o image:icon-edit-pencil-blue-outline.png["Icona Edit (Modifica)"] sotto *NFS*.
. Selezionare la check box *Enable NFS*.
. Abilitare o disabilitare un protocollo NFS:
+
** Per abilitare NFSv3, selezionare la check box *Abilita supporto versione 3* e valutare la seguente opzione:
+
*** Seleziona la check box *Abilita FSID e identificatori di file a 64 bit* per abilitare il supporto per ID di file system e identificatori di file a 64 bit per NFSv3.


** Per disabilitare NFSv3, deselezionare la check box *Abilita supporto versione 3*.
** Per abilitare NFSv4.0, selezionare la check box *Abilita supporto versione 4* e valutare le seguenti opzioni:
+
*** Seleziona la check box *Enable ACL* per consentire l'utilizzo di un access control list.
*** Seleziona la check box *Abilita delega di lettura* per consentire l'accesso in sola lettura.
*** Seleziona la check box *Abilita delega di scrittura* per consentire l'accesso in scrittura.
*** Seleziona la check box *Abilita FSID e identificatori di file a 64 bit* per abilitare il supporto per ID di file system e identificatori di file a 64 bit. Questa check box si applica a NFSv4.0, NFSv4.1 e NFSv4.2. Questa è distinta dalla check box utilizzata per controllare il supporto a 64 bit per NFSv3.


** Per disabilitare NFSv4.0, deselezionare la check box *Abilita supporto versione 4*.
** Per abilitare NFSv4.1, selezionare la check box *Abilita supporto versione 4.1 e 4.2* e quindi selezionare la check box *Abilita supporto versione 4.1*. Considerare le seguenti opzioni:
+
*** Seleziona la check box *Enable ACL* per consentire l'utilizzo di un access control list.
*** Seleziona la check box *Abilita delega di lettura* per consentire l'accesso in sola lettura.
*** Seleziona la check box *Abilita delega di scrittura* per consentire l'accesso in scrittura.
*** Seleziona la check box *pNFS* per consentire ai client di utilizzare parallel NFS (pNFS) per prestazioni migliori.
*** Seleziona la check box *Abilita FSID e identificatori di file a 64 bit* per abilitare il supporto per ID di file system e identificatori di file a 64 bit. Questa check box si applica a NFSv4.0, NFSv4.1 e NFSv4.2. Questa è distinta dalla check box utilizzata per controllare il supporto a 64 bit per NFSv3.


** Per disabilitare NFSv4.1, deselezionare la check box *Abilita supporto versione 4.1*.
** Per abilitare NFSv4.2, selezionare la check box *Abilita supporto versione 4.1 e 4.2* e quindi selezionare la check box *Abilita supporto versione 4.2*. Considerare le seguenti opzioni:
+
*** Seleziona la check box *Enable ACL* per consentire l'utilizzo di un access control list.
*** Seleziona la check box *Abilita delega di lettura* per consentire l'accesso in sola lettura.
*** Seleziona la check box *Abilita delega di scrittura* per consentire l'accesso in scrittura.
*** Seleziona la check box *pNFS* per consentire ai client di utilizzare parallel NFS (pNFS) per prestazioni migliori.
*** Seleziona la check box *Abilita FSID e identificatori di file a 64 bit* per abilitare il supporto per ID di file system e identificatori di file a 64 bit. Questa check box si applica a NFSv4.0, NFSv4.1 e NFSv4.2. Questa è distinta dalla check box utilizzata per controllare il supporto a 64 bit per NFSv3.


** Per disabilitare NFSv4.2, deselezionare la check box *Abilita supporto versione 4.2*.


. Facoltativamente, è possibile specificare il *domain name V4* per il protocollo NFS.
. Selezionare *Salva*.
