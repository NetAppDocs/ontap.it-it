= Processo interrotto a causa dell'arresto
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Job aborted due to shutdown`

* *Causa*
+
Riavvio del sistema storage in corso.

* *Azione correttiva*
+
Riprovare a eseguire il processo dopo il riavvio del sistema di storage.
