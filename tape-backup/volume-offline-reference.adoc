= Volume offline
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Volume offline`

* *Causa*
+
Il volume di cui viene eseguito il backup non è in linea.

* *Azione correttiva*
+
Portare il volume online e riprovare il backup.
