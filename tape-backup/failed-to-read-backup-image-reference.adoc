= Impossibile leggere l'immagine di backup
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Failed to read backup image`

* *Causa*
+
Il nastro è corrotto.

* *Azione correttiva*
+
Se il nastro è corrotto, non è possibile eseguire l'operazione di ripristino.
