= Nessun file creato
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`No files were created`

* *Causa*
+
È stato tentato un DAR di directory senza abilitare la funzionalità DAR avanzata.

* *Azione correttiva*
+
Abilitare la funzionalità DAR avanzata e riprovare a eseguire il DAR.
