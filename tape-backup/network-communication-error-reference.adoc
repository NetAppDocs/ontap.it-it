= Errore di comunicazione di rete
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Network communication error`

* *Causa*
+
La comunicazione con un nastro remoto in una connessione NDMP a tre vie non è riuscita.

* *Azione correttiva*
+
Verificare la connessione di rete al telecomando.
