= Impossibile creare lo snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Failed to create snapshot`

* *Causa*
+
Il volume contiene già il numero massimo di copie Snapshot.

* *Azione correttiva*
+
Eliminare alcune copie Snapshot, quindi riprovare l'operazione di backup.
