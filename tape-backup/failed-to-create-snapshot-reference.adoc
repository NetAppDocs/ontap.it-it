= Impossibile creare lo snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Failed to create snapshot`

* *Causa*
+
Il volume contiene già il numero massimo di snapshot.

* *Azione correttiva*
+
Eliminare alcuni snapshot e riprovare l'operazione di backup.
