= ID_sessione ndmpd non attivo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`ndmpd session session_ID not active`

* *Causa*
+
La sessione NDMP potrebbe non esistere.

* *Azione correttiva*
+
Utilizzare `ndmpd status` Per visualizzare le sessioni NDMP attive.
