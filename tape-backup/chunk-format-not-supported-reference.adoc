= Formato chunk non supportato
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Chunk format not supported`

* *Causa*
+
L'immagine di backup non è di SMTape.

* *Azione correttiva*
+
Se l'immagine di backup non è SMTape, riprovare l'operazione con un nastro che dispone del backup SMTape.
