= Il qtree di destinazione è di sola lettura
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Destination qtree is read-only`

* *Causa*
+
Il qtree su cui si tenta di eseguire il ripristino è di sola lettura.

* *Azione correttiva*
+
Provare a ripristinare i dati in un'altra posizione.
