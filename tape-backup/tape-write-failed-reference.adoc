= Scrittura del nastro non riuscita
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Tape write failed`

* *Causa*
+
Il nastro utilizzato per il backup è danneggiato.

* *Azione correttiva*
+
Sostituire il nastro e riprovare a eseguire il processo di backup.
