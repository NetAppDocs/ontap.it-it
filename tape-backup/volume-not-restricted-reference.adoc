= Volume non limitato
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Volume not restricted`

* *Causa*
+
Il volume di destinazione in cui vengono ripristinati i dati non è limitato.

* *Azione correttiva*
+
Limitare il volume e riprovare l'operazione di ripristino.
