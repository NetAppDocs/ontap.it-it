= Blocchi fuori servizio
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Chunks out of order`

* *Causa*
+
I nastri di backup non vengono ripristinati nella sequenza corretta.

* *Azione correttiva*
+
Ripetere l'operazione di ripristino e caricare i nastri nella sequenza corretta.
