= Impossibile ottenere l'ultimo snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Failed to get latest snapshot`

* *Causa*
+
L'ultimo snapshot potrebbe non esistere perché il volume viene inizializzato da SnapMirror.

* *Azione correttiva*
+
Riprovare al termine dell'inizializzazione.
