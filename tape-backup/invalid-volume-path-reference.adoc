= Percorso del volume non valido
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Invalid volume path`

* *Causa*
+
Il volume specificato per l'operazione di backup o ripristino non viene trovato.

* *Azione correttiva*
+
Riprovare a eseguire il processo con un percorso del volume e un nome del volume validi.
