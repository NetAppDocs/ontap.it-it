= Errore di supporto in scrittura su nastro
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Media error on tape write`

* *Causa*
+
Il nastro utilizzato per il backup è danneggiato.

* *Azione correttiva*
+
Sostituire il nastro e riprovare a eseguire il processo di backup.
