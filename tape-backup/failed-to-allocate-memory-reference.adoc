= Impossibile allocare la memoria
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Failed to allocate memory`

* *Causa*
+
La memoria del sistema è esaurita.

* *Azione correttiva*
+
Riprovare a eseguire il processo in un secondo momento quando il sistema non è troppo occupato.
