= Asserzione interna
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Messaggio*
+
`Internal assertion`

* *Causa*
+
Si è verificato un errore interno SMTape.

* *Azione correttiva*
+
Notificare l'errore e inviare il `etc/log/backup` file al supporto tecnico.
