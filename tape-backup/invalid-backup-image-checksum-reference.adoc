= Checksum immagine di backup non valido
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Messaggio*
+
`Invalid backup image checksum`

* *Causa*
+
Il nastro è corrotto.

* *Azione correttiva*
+
Se il nastro è corrotto, non è possibile eseguire l'operazione di ripristino.
