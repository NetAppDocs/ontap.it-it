= Panoramica sulla gestione della rete
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
È possibile utilizzare le seguenti informazioni per eseguire l'amministrazione di base della rete di storage. È possibile configurare le porte di rete fisiche e virtuali (VLAN e gruppi di interfacce), creare LIF utilizzando IPv4 e IPv6, gestire i servizi di routing e risoluzione degli host nei cluster, utilizzare il bilanciamento del carico per ottimizzare il traffico di rete e monitorare un cluster utilizzando SNMP.

Se non diversamente specificato, queste procedure si applicano a tutte le versioni di ONTAP 9.

Attenersi alle seguenti procedure nei seguenti casi:

* Vuoi conoscere la gamma di funzionalità di gestione della rete ONTAP.
* Si desidera utilizzare la CLI, non System Manager.
