= Elimina gli SVM attivo-attivo prima di eseguire il revert da ONTAP 9.19.1
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Il multipathing attivo-attivo SAN non è supportato in ONTAP 9.18.1 e versioni precedenti. Prima di tornare da ONTAP 9.19.1, devi eliminare tutte le storage virtual machine (SVM) con configurazioni attivo-attivo.

.Fasi
. Identifica gli SVM che hanno l'opzione san-multipathing configurata per active-active o active-active-scsi-only:
+
[source, cli]
----
vserver show
----
. Elimina tutti gli SVM configurati per il multipathing active-active o active-active-solo SCSI:
+
[source, cli]
----
vserver delete -vserver <SVM>
----
. Verifica che gli SVM siano stati eliminati:
+
[source, cli]
----
vserver show
----
