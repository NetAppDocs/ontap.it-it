= Rimuovere le seconde chiavi di accesso S3 prima di eseguire il revert da ONTAP 9.19.1
:allow-uri-read: 


[role="lead"]
Se hai eseguito l'aggiornamento a ONTAP 9.19.1 e hai aggiunto una seconda chiave di accesso S3 a un utente, queste chiavi devono essere rimosse prima di poter tornare a una versione precedente di ONTAP.

.Fasi
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["Mostra le chiavi di accesso S3"] per determinare se sono state aggiunte seconde chiavi di accesso per gli utenti S3.
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["Rimuovere le seconde chiavi di accesso"] per tutti gli utenti S3.
