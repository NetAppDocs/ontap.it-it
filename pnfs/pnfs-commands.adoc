= Comandi, statistiche e registri eventi pNFS
:allow-uri-read: 


[role="lead"]
Questi comandi CLI ONTAP riguardano specificamente pNFS. È possibile utilizzarli per configurare, risolvere problemi e raccogliere statistiche.



== Abilitare NFSv4.1

[listing]
----
nfs modify -vserver SVM -v4.1 enabled
----


== Abilitare pNFS

[listing]
----
nfs modify -vserver SVM -v4.1-pnfs enabled
----

== Cattura contatori di prestazioni specifici di pNFS (privilegi avanzati)

[listing]
----
statistics start -object nfsv4_1 -vserver SVM -sample-id [optional-name]
----


== Visualizza i contatori delle prestazioni specifici di pNFS (privilegi avanzati)

[listing]
----
statistics show -object nfsv4_1 -vserver SVM
----

== Visualizza le informazioni sulla versione NFS per i client connessi

È anche possibile visualizzare le connessioni NFS con `nfs connected-clients show` comando. Tieni presente che l'elenco dei client visualizzati comprende i client che hanno avuto traffico NFS attivo nelle ultime 48 ore. I client NFS inattivi (anche se ancora montati) potrebbero non essere visualizzati finché non si accede al montaggio. È possibile filtrarli per mostrare solo i client a cui è stato effettuato l'accesso più di recente specificando `-idle-time` caratteristica.

Ad esempio, per visualizzare i client con attività negli ultimi 10 minuti per pNFS SVM:

[listing]
----
cluster::*> nfs connected-clients show -vserver PNFS -idle-time <10m>
